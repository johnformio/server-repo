Getting started with your app

Form.IO provides the forms and resource structure behind your custom application. Start by creating the Resources, and then create the free-form Forms that your application needs.

Once you have created the Resources and Forms, you can embed them within your application using the <formio> Angular.js directive. It works just like an <img> tag, but used for Form I/O.

It is recommended to walk through the tutorial, which provides a step-by-step tuturial on how to build an application with Form.IO.
