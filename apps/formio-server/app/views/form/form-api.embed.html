Embedding your form

There are two different ways to embed this form. If you are utilizing Angular.js in your application, then you can embed it using the Angular.js directive. Otherwise, you can use the iframe method of form embedding.

Angular.js Integration

Angular.js integration can be accomplished by first including the JavaScript src in your HTML application.

<script src="http://form.io/bower_components/formio/dist/formio.min.js" type="text/javascript"></script>

Now, within your HTML source, you can use the <formio> directive like so. 

<formio src="'/app/{{ appId }}/form/{{ form._id }}'"></formio>
