<form.io> Main Page

Create and manage multiple applications

The Form.IO platform allows you to create multiple applications and development teams to help manage your applications in one easy place.

Amazing Form Builder

Form.IO provides a state-of-the-art form builder that allows you to easily drag & drop your form components to create both the Form and API!

Create Resources

with corresponding Form's and API's

Each Resource is an object represented within your application. You can use a simple drag & drop form builder to easily create your Resource forms and the API's to go with them.

Create Forms

Easily create forms by reusing Resource fields. Form.IO allows you to rapidly create forms that reference Resource information by dragging and dropping the Resource fields onto the form. Building form interfaces for your resources has never been easier!

Customize Behavior with Actions

Form.IO's action system allows for complex functionality to be attached to the form submission. From Emails to User Authentication, anything is possible!

Embed form.io into your Application.

Form.IO's form embedding technology allows you to utilize one single <formio> tag to embed the form in your application. It's as easy as an <img> tag! You can even directly theme the forms using simple CSS.
