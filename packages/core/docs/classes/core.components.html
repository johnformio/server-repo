Static importComponent

  • importComponent(definition: any): any
    • Defined in src/core/index.ts:81

    Takes a component JSON definition, and then converts it into a Component class that can be instantiated as well as created with createComponent.

    For example, you can import an H3 component using the following code.

    Components.importComponent({
   type: 'h3',
   extends: 'html',
   schema: {
     tag: 'h3'
   }
});
const h3 = Components.createComponent({
   type: 'h3',
   content: 'This is an H3 header!'
});
console.log(h3.render()); // Outputs "<h3>This is an H3 header!</h3>"

    • definition: any

    Returns any
